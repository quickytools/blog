Effortless Themes

Are you tired

of pushing pixels because you don’t have a rock-star designer who dreams in screens? Thank Google for organizing Material Design principles and delivering a fully developed design system!

  1. Download and install Sketch (free trial lasts for 30 days)
  2. Download and install Material Plugin for Sketch (free)
  3. Create a base theme for effortlessly crafting your screens (quick overview)
    1. Save this theme (Sketch file) and set it as a library for use by other Sketch files
    2. For best practices keep the page structure as is and create new pages for new/modified components. The first 5 pages were generated by the plugin. The 6th page quickytools Components contains new and modified symbols. Add more pages for organization as necessary.

Symbol/component override

  1. Create a new Sketch file for crafting screens
  2. Create a new artboard. Keyboard shortcut should be a. Predefined sizes can be selected in the inspector panel at the right.
  3. Insert an app bar by selecting Insert>Symbols>custom theme>App Bar>Top App Bar>Regular>Raised>Primary Color
    1. Select the app bar to bring up its details in the inspector panel. Open Overrides>ICONS>Icon 1 and hover over the custom theme to expand the icon options. Notice the selection is limited. This is due to the base theme not generating the entire material icon set.
  4. Open Plugins>Material>Open Theme Editor... and select the pencil icon to open the custom theme Sketch file
    1. Select the fourth icon (four dots) which lists the available icon sets
    2. Select any of the sets to open a web browser and download the set as a Sketch file
    3. Add the newly downloaded icon set file as a Sketch library
  5. Go back to the Sketch file in in step 3.1 and open Overrides>ICONS>Icon 1 but instead of hovering over the custom theme hover over the downloaded icon set and select an icon not a part of the generated custom theme
  6. Change the color of the icon with a preset color by opening Overrides>ICONS>Icon 1>Color and selecting a preset color from the custom theme
    1. If the Primary or Secondary color is selected the secondary color can by set under Tonal Palette
