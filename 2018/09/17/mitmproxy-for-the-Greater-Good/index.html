mitmproxy on Sunday

Some days are for being lazy

and plenty of code is not debuggable. Rather than refactoring inspection+hack often gets the job done. Here we trace network activty with mitmproxy

  1. On OSX install mitmproxy with brew
  2. Run mitmdump on a specific port mitmdump --server --listen-port 8081
  3. Configure the client
    a. Turn off data
    b. Edit the WIFI connection to use the mitm machine as a proxy
  4. Normal sites should work but for secure sites visit mitm.it on the client (follow instructions)
    a. Download the appropriate cert listed
    b. Install the cert proper (after removing previous certs)
  5. 👁 💻

Scripting

  1. Install python3 and pip3
  2. Install mitmproxy python module pip3 install mitmproxy
  3. Create script for printing requests/responses
  4. Run the script mitmdump --server --listen-port 8081 -s dump.py
  5. ✌
